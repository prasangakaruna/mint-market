# üöÄ Quick Start Guide

Get the Mint Market app running in 5 minutes!

## Step 1: Install Dependencies (2 minutes)

Open your terminal in the project folder and run:

```bash
npm install
```

Wait for all packages to download and install.

## Step 2: Start the App (1 minute)

```bash
npm start
```

This opens the Expo Dev Tools in your browser.

## Step 3: Run on Your Phone (2 minutes)

### iPhone or Android:

1. **Download Expo Go app**:
   - iPhone: Search "Expo Go" in App Store
   - Android: Search "Expo Go" in Play Store

2. **Scan the QR Code**:
   - iPhone: Use Camera app to scan the QR code
   - Android: Use Expo Go app to scan the QR code

3. **Wait for the app to load** (first time takes ~30 seconds)

## Common Issues

### "Can't scan QR code?"
- Make sure you have Expo Go installed
- Try refreshing the page
- Make sure phone and computer are on same WiFi

### "App won't load?"
- Check your internet connection
- Try running `npm start -c` to clear cache
- Restart Expo Dev Tools

### "npm: command not found"
- You need to install Node.js first
- Download from: https://nodejs.org/
