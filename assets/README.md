# Assets Directory

This directory contains the app's static assets including icons, splash screens, and images.

## Required Assets

To complete the app setup, you need to add the following image files to this directory:

### App Icons
- `icon.png` - App icon (1024x1024 px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024 px)
- `favicon.png` - Web favicon (48x48 px or larger)

### Splash Screen
- `splash.png` - Splash screen image (1242x2436 px)

## Creating Assets

You can create these assets using:
1. Design tools like Figma, Adobe XD, or Canva
2. Online icon generators
3. The Mint Market logo/branding

## Asset Specifications

### icon.png
- Size: 1024x1024 pixels
- Format: PNG with transparency
- Purpose: Main app icon for iOS and Android

### adaptive-icon.png
- Size: 1024x1024 pixels
- Format: PNG with transparency
- Safe zone: Content should be within 768x768 center area
- Purpose: Android adaptive icon

### splash.png
- Size: 1242x2436 pixels (or larger)
- Format: PNG
- Background: White (#ffffff) or your brand color
- Purpose: Loading screen when app starts

### favicon.png
- Size: 48x48 pixels or larger
- Format: PNG with transparency
- Purpose: Web browser tab icon

## Temporary Placeholders

Until you add proper assets, you can:
1. Use the default Expo assets
2. Generate simple colored icons using online tools
3. Use text-based icons temporarily
